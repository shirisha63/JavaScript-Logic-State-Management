# 📝 To-Do List Application

## 📌 Project Overview

This is a simple and interactive **To-Do List Application** developed using **HTML, CSS, and JavaScript**. The application allows users to manage their daily tasks efficiently with complete CRUD functionality and automatic data storage using the browser's Local Storage.

The project demonstrates JavaScript DOM manipulation, event handling, client-side state management, and local data persistence.

---

## 🚀 Features

* ✅ Add new tasks
* ✅ Edit existing tasks
* ✅ Delete tasks
* ✅ Mark tasks as completed
* ✅ Filter tasks:

  * All
  * Active
  * Completed
* ✅ Automatically save tasks using `window.localStorage`
* ✅ Retain tasks after browser refresh
* ✅ Responsive and user-friendly interface

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (if not already installed).
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
6. Start adding and managing your tasks.

---

## 💻 Functionality

### Create

Add a new task using the input field.

### Read

Display all saved tasks on the screen.

### Update

Edit an existing task.

### Delete

Remove a task from the list.

### Complete

Mark tasks as completed using the checkbox.

### Local Storage

Tasks are automatically saved and restored after refreshing the browser.

### Filters

* **All** – Displays all tasks.
* **Active** – Displays only incomplete tasks.
* **Completed** – Displays only completed tasks.
